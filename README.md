# chatGPTNext

## Demo app to show the easy integration between OpenAI's chatGPT AI

### Check it on: https://chatgpt-steel-six.vercel.app/

---

## Features

- API call with a prompt that returns a response
- Dark Mode / Light Mode
- Beautiful Design
